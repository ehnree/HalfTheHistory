In order to view site:
	1) CMD+space "Terminal"
	2) Type "hthsite"
	3) Type "show"
	2) Open a browser
	3) Type "localhost:4000" in address bar

To get changes:
	1) git pull


Instructions for starting web server

1) Fork/Clone Repo
2) gem install bundler
3) bundle install
4) update _config.yml

running server
1) bundle exec jekyll build
2) bundle exec jekyll serve

development
1) bundle exec jekyll serve --config _config.yml,_config_dev.yml
